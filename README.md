# MAD2 project

Here I will keep my code of MAD2 project I'll make in the bootcamp.

Create virtual environment with name "venv" by running below command:
 python -m venv venv

Activate:
Windows: venv/Scripts/activate
linux: source venv/bin/activate

install requirements
pip install -r .\requirements.txt

if installed new packages then update requirements by
pip freeze > requirements.txt
