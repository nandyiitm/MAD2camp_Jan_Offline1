# 🚀 MAD2 Project

This repository contains my **MAD2 project** developed during the bootcamp.
It includes all source code, dependencies, and setup instructions required to run the project locally.

---

### commands to run backend
  cd backend
  python -m venv venv // to create virtual environment
  // activate virtaul environment
    venv/Scripts/activate // activate venv in win
    source venv/bin/activate // activate venv in linux
  pip install -r requirements.txt
  python app.py

### commands to run frontend
   cd frontend
   npm install
   npm run dev
